Column Flex

Create a column width defined size.

Arguments

  • int $col: Number of columns (default 12)
  • int $sum: Total columns
  • bool/string $with-gutter: true = add gutter, false = no gutter, 'left' = only left gutter, right = only right gutter

Sur certains projets, nous devons gérer des débordements de fonds en dehors du container). Cette pratique n'est pas toujours simple à gérer. Pour toutes questions, demander à Tonio.
Cette mixin permet donc de créer dynamiquement ce débordement. Le principe est simple :
- Le premier paramètre consiste à préciser si le débordement provient de la gauche ou de la droite
- le second est le breakpoint à partir duquel on veut que ce débordement s'applique
- le troisième peut à la fois être une chaîne de caractère (deux arguments) ou un nombre (un argument). Le premier argument fait référence au nombre de colonnes sur lesquelles le débordement va s'étendre. Le deuxième (facultatif) est une valeur en pixel pour ajouter un décalage supplémentaire au nombre de colonnes (4 colonnes + 300px par exemple).
- le quatrième représente la variable d'une valeur en px de la distance hors container (pour ne pas avoir un bloc qui s'étend du bord de l'écran au container sur des grandes résolutions)

Un bon exemple est sur le projet Culligan, sur la home, sur le bloc "Avec Culligan, protégez la planète !"
