Breakpoints

mixin breakpoint use predefined

// Min-width div { @include breakpoint(xsm) { } => mobile-landscape @include breakpoint(sm) { } => tablet-portrait @include breakpoint(md) { } => tablet-landscape @include breakpoint(lg) { } => desktop @include breakpoint(xlg) { } => desktop large } // Max-width div { @include breakpoint-max(xsm) { } => mobile-landscape ... } // Only one breakpoint div { @include breakpoint-only(sm) { } => only tablet-portrait ... } // Bewteen 2 breakpoints div { @include breakpoint-between(sm, lg) { } => tablet-portrait => desktop large ... }

Container

Arguments

  • bool $with-gutter: Add gutter (default false)
.container { @include container(); @include container-max-widths(); } .container-fluid { @include container-fluid(); }

Gutterize

Add default gutter for all defined breakpoints

.box { @include gutterize; } //only gutter on left side .box { @include gutterize(true, false); } //only gutter on right side .box { @include gutterize(false, true); }

Column Flex

Create a column width defined size.

Arguments

  • int $col: Number of columns (default 12)
  • int $sum: Total columns
  • bool/string $with-gutter: true = add gutter, false = no gutter, 'left' = only left gutter, right = only right gutter
//For the parent to remove margin .row{ @include flex-row(); } //with gutter .box { @include flex-col(2); } //no gutter .box { @include flex-col(2, 12, false); } //left gutter .box { @include flex-col(2, 12, 'left'); } //right gutter .box { @include flex-col(2, 12, 'right'); }

Column (without flex)

Create a column width defined size.

Arguments

  • int $col: Number of columns (default 12)
  • int $sum: Total columns
  • bool/string $with-gutter: true = add gutter, false = no gutter, 'left' = only left gutter, right = only right gutter
  • bool $float: true or false (default false)
// with gutter .box { @include column(2); } ...

Offset

Pull block from a defined column size

Arguments

  • int $col: Number of columns
  • int $sum: Total columns
  • string $direction: left or right (default left)
.box { @include offset(2, 12); }
